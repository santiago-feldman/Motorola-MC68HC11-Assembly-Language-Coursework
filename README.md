# Motorola-MC68HC11-Assembly-Language-Coursework
## Introduction to Computing (Motorola MC68HC11 Assembly Language Programming) | ITBA

This repository contains the practical assignments done for the "22.06 Introduction to computing" subject at Instituto Tecnológico de Buenos Aires:

- **TP1 | Logic and Boolean algebra** (not included)
- **TP2 | Addressing modes, Introduction to programming, Stack and Subroutines**
- **TP3 | Arrays and Data structures**
- **TP4 | Operating systems, Hard disks, Filesystems and PC technology**
- **TP5 | Display**
- **TP6 | Calculator**
- **TPE | Battleship game**
- **TP NaN | Hard disks, Filesystems, Intel 8088/86, Interruptions and Floating-point** (not included)
